# IPL-Data-Analysis
Analysis of IPL matches and player performance using Python

This project analyzes IPL cricket data using Python.

## Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis
- Team win percentage
- Toss decision impact
- Player of the match analysis
- Season performance

## Dataset
IPL Matches and Deliveries dataset
